# PROYECTO-INTEGRAL-QA
Proyecto Final QA - Urban Scooter

## Descripción del Proyecto

Este proyecto representa una evaluación integral de Quality Assurance para **Urban Scooter**, una aplicación de alquiler de scooters eléctricos. El trabajo demuestra competencias  en análisis, diseño y ejecución de pruebas de software a través de múltiples plataformas y metodologías.

## Objetivo

Validar la funcionalidad, usabilidad y confiabilidad de la aplicación Urban Scooter mediante pruebas exhaustivas que aseguren una experiencia de usuario óptima y el correcto funcionamiento del sistema.

## Alcance del Proyecto

### **Análisis de Requisitos y Diseño de Pruebas**
- Análisis detallado de especificaciones funcionales
- Creación de mapas mentales para visualización de requisitos
- Diseño de casos de prueba utilizando técnicas de partición de equivalencia y análisis de valores límite
- Desarrollo de matrices de trazabilidad

### **Testing de Aplicación Web**
- Pruebas funcionales de la interfaz web
- Validación de formularios y flujos de usuario
- Verificación de compatibilidad entre navegadores
- Testing de responsividad y usabilidad

### **Testing de Aplicación Móvil**
- Pruebas de funcionalidad en dispositivos móviles
- Validación de gestos táctiles y navegación
- Testing de rendimiento y comportamiento en diferentes resoluciones
- Verificación de la experiencia de usuario móvil

### **Testing de API**
- Validación de endpoints y métodos HTTP
- Verificación de estructura de datos y respuestas
- Testing de códigos de estado y manejo de errores
- Documentación de comportamiento de servicios backend

## Metodología

- **Enfoque:** Testing manual sistemático
- **Técnicas:** Caja negra, partición de equivalencia, análisis de valores límite
- **Documentación:** Casos de prueba detallados, reportes de defectos, matrices de cobertura
- **Herramientas:** Navegadores web, dispositivos móviles, herramientas de testing de API

## Entregables

- Documentación completa de casos de prueba
- Reportes detallados de defectos encontrados
- Mapas mentales de análisis de requisitos
- Matrices de trazabilidad y cobertura
- Recomendaciones de mejora
